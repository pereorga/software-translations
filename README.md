# software-translations

A few Catalan translations of software projects, mostly open-source, to be used
later in https://github.com/Softcatala/translation-memory-tools/.

## Requirements

* Bash (modern versions)
* csvkit
* dos2unix
* gettext
* Translate Toolkit
* unzip

## Installation

On Ubuntu/Debian:

```
sudo apt-get install csvkit dos2unix gettext translate-toolkit unzip
```

On Mac:
```
brew install bash csvkit dos2unix gettext translate-toolkit unzip
```
